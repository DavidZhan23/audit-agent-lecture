# 04 · nn — ③ 超多特征与 ANN

- **锚点：** `#nn`
- **侧栏：** 超多特征与 ANN（15′）
- **课纲定位：** 特征极多、难以人工造齐时，用 ANN 学表示

## 教学目标

- ANN 仍属 ML；面向高维输入
- 看见数字 / 人脸特征 ≠ 理解制度
- 优劣势：少手工特征 / 数据算力与可解释性代价

## 内容块

1. `Definition` — ANN
2. `DatasetAnchor` — BX-42306：票面286、平台86，疑似数字2后加 / 方程说明
3. `DigitsImageLab` — 8×8 教学子集（经典小网络）
4. **趣味支线 `FacePredictLab`** — 真实 ResNet18 人脸三分类（上传 / 摄像头），用于说明“像素→多层特征→分类概率”的迁移，不作为审计主线证据
5. `InlinePythonLab` neural
6. 训练循环 / 留下的矩阵
7. `CapabilityBoundary` + 优劣势
8. `DeepDive`（反向传播等）
9. `Bridge` → LLM

## 人脸演示依赖

- 后端：`services/face-predict`（FastAPI，默认 `:8765`）
- 前端代理：`POST /api/face-predict` → Python 服务
- 前端会压缩图片（避免线上 413）；非 HTTPS 公网 IP 下实时摄像头不可用，改用上传 / `capture` 拍照
- 说明：`services/face-predict/README.md`
- 教学边界：置信度不等于事实；需提示偏差、隐私、授权和使用范围
